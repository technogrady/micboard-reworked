Keyboard Shortcuts

  • ? - Show keyboard shortcuts
  • 0 - Show all slots
  • 1...9 - Load group
  • d - Start demo mode
  • e - Open group editor
  • t - Toggle TV view
  • i - Change tv display mode
  • f - Toggle fullscreen
  • g - Toggle image backgrounds
  • v - Toggle video backgrounds
  • n - Extended Name editor
  • s - Device configuration editor
  • p - Planning Center mic assignments
  • q - Show QR code
  • esc - reload micboard

Every shortcut is also a button in the top toolbar. Click ☰ in the navbar to hide or show the buttons. The toolbar hides itself in TV and fullscreen modes — use the faint ☰ button in the bottom-right corner to bring it back.
